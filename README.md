# Mars Rover Platformer

Be a mars rover in this platformer in python. Use WASD or arrow keys to move.


# Features

- Parralax background
- Fun pixel art
- Spikes
- Coyote time
- Ice
- Springs
- Checkpoints
- Sand


# Setup

- Install python
- Download repo
- Run this in the terminal where you installed the repo:

~~~bash

pip install -r requirements.txt

~~~

- Run main.py
